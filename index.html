ThingSet SDK DocumentationÔÉÅ

ThingSet is a transport-agnostic and self-explanatory remote API for embedded devices and humans.

The ThingSet software development (SDK) for Zephyr leverages the Zephyr RTOS APIs to provide ready-to-use implementations for multiple different interfaces / subsystems (see navigation at the left).

Further interfaces like MQTT and WebSocket over WiFi are currently under development.

The ThingSet node library is used internally for data processing.

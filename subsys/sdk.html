SDK

Configuration Options

The ThingSet SDK itself has to be enabled for all subsystems and interfaces:

  • CONFIG_THINGSET_SDK

Below options define the general publication settings:

  • CONFIG_THINGSET_SUBSET_LIVE_METRICS

  • CONFIG_THINGSET_REPORTING_LIVE_ENABLE_PRESET

  • CONFIG_THINGSET_REPORTING_LIVE_PERIOD_PRESET

  • CONFIG_THINGSET_SUBSET_SUMMARY_METRICS

  • CONFIG_THINGSET_REPORTING_SUMMARY_ENABLE_PRESET

  • CONFIG_THINGSET_REPORTING_SUMMARY_PERIOD_PRESET

Common options for the SDK:

  • CONFIG_THINGSET_GENERATE_NODE_ID

  • CONFIG_THINGSET_SHARED_TX_BUF_SIZE

  • CONFIG_THINGSET_SDK_THREAD_STACK_SIZE

  • CONFIG_THINGSET_SDK_THREAD_PRIORITY
